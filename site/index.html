Core Output

Raw perception, not opinions

Every frame, agentic-sense outputs structured data about every face in view. No "happy" or "sad" labels — just precise geometric and muscular measurements. Your agent decides what they mean.

  "faceCount": 1,
  "faces": [{

478 Landmarks
Full face mesh with iris tracking. Head pose, eye corners, lip contour — sub-pixel precision at 30fps.
🎭
52 Blendshapes
Every facial muscle as a 0-1 weight. jawOpen, mouthSmile, browDown, eyeSquint — the raw signal for expression analysis.
📐
Geometric Metrics
EAR (eye aspect ratio), IPD (inter-pupillary distance), mouth ratio, head yaw/pitch/roll — computed per frame.
👥
Multi-Face
Track up to 3 faces simultaneously. Each face gets its own complete data struct.
🔒
Fully Local
Zero network requests after load. MediaPipe WASM runs in-browser. Camera feed never leaves the device.
📦
~14MB Total
WASM engine + face model + your code. No npm install, no build step. One import and you're sensing.

Quick Start

Three lines to perceive

// 1. Import
import { SenseEngine } from './engine.js'
import { extractFrame } from './sense-data.js'

// 2. Init
const engine = new SenseEngine(videoEl, canvasEl)
await engine.init()

// 3. Sense
function loop() {
  const result = engine.detect()
  if (result.sense) {
    // result.sense.faces[0].head.facing → boolean
    // result.sense.faces[0].eyes.avgEAR → blink state
    // result.sense.faces[0].blendshapes.smileL → 0-1
  }
  requestAnimationFrame(loop)
}
